# Sistem Informasi Klinik (PBO Project)

Aplikasi manajemen klinik sederhana berbasis Java Desktop (NetBeans) dan MySQL untuk memenuhi tugas praktikum Pemrograman Berorientasi Objek.

## Fitur Utama
- **Autentikasi:** Sistem login untuk admin.
- **CRUD Pasien:** Tambah, Tampil, Ubah, Hapus, dan Pencarian Pasien.
- **Manajemen Data Dokter:** Kelola data dokter spesialis.
- **Manajemen Data Perawat:** Kelola data perawat klinik.

## Teknologi
- **IDE:** Apache NetBeans
- **Language:** Java 21
- **Build Tool:** Maven
- **Database:** MySQL (mysql-connector-j)

## Cara Menjalankan
1. Pastikan Anda memiliki XAMPP/MySQL terinstal.
2. Import file `db_klinik.sql` yang tersedia di folder root ke phpMyAdmin.
3. Buka proyek ini di Apache NetBeans.
4. Pastikan library `mysql-connector-j` sudah terunduh via Maven.
5. Jalankan `AppKlinik.java`.

## Dokumentasi Database
File `db_klinik.sql` berisi struktur tabel dan data sampel untuk pengujian aplikasi.
